# Projeto: Validador de Idade em Python

Este programa solicita o nome completo e o ano de nascimento do usuário, realizando a validação dos dados e calculando a idade com base no ano de 2022.

## Funcionalidades
- **Entrada de Nome:** Recebe o nome completo do usuário.
- - **Validação de Ano:** O ano de nascimento deve ser um número inteiro entre **1922** e **2021**.
  - - **Tratamento de Erros:** Caso o usuário digite um valor não numérico ou um ano fora do intervalo permitido, o sistema exibe uma mensagem de erro e solicita a informação novamente.
    - - **Cálculo de Idade:** Exibe o nome e a idade que o usuário completou ou completará no ano de 2022.
     
      - ## Como executar
      - 1. Certifique-se de ter o Python instalado.
        2. 2. Execute o arquivo:
           3. ```bash
              python main.py
              ```
